Bias Intensity Regressor Metrics

Exhaustive continuous regression analysis of the fine-tuned DistilBERT Regressor, measuring absolute error bounds and correlation bounds for predicting the exact percentage of bias (0.0 to 1.0).

Regression Metric Value Interpretation / Goal
Mean Squared Error (MSE) {{ classifier_metrics.intensity_regression_metrics.eval_mse | round(5) }} Average squared prediction error. Closer to 0 is better.
Mean Absolute Error (MAE) {{ classifier_metrics.intensity_regression_metrics.eval_mae | round(5) }} Average magnitude of prediction mistakes. Lower is better.
Root Mean Squared Error (RMSE) {{ classifier_metrics.intensity_regression_metrics.eval_rmse | round(5) }} Standard deviation of the residuals. Heavily penalizes large outliers.
R² Coefficient of Determination {{ classifier_metrics.intensity_regression_metrics.eval_r2 | round(5) }} Proportion of variance explained by the model. 1.0 is perfect fit.
Mean Absolute Percentage Error (MAPE) {{ classifier_metrics.intensity_regression_metrics.eval_mape | round(5) }} Relative error margin in percentages.
Median Absolute Error {{ classifier_metrics.intensity_regression_metrics.eval_median_absolute_error | round(5) }} Robust to anomalous outliers. Lower is better.
Explained Variance Score {{ classifier_metrics.intensity_regression_metrics.eval_explained_variance | round(5) }} Measures system dispersion explanation capacity.
Max Absolute Error {{ classifier_metrics.intensity_regression_metrics.eval_max_error | round(5) }} Worst-case single error across the validation set. Lower is better.
Pearson Correlation Coefficient (r) {{ classifier_metrics.intensity_regression_metrics.eval_pearson_r | round(5) }} Linear correlation strength. Close to 1.0 represents perfect sync.
Spearman Rank Correlation (ρ) {{ classifier_metrics.intensity_regression_metrics.eval_spearman_rho | round(5) }} Monotonic relationship strength. Highly robust to non-linear shifts.

Interpreting Class Confusions

Our custom DistilBERT multi-class classifier exhibits high precision in separating explicit biases from neutral scenarios. Some overlapping biases (e.g., Social Class Bias and Professional/Educational Bias) represent minor off-diagonal clusters, which is expected due to their semantic linkages.

Active Mitigation in Place

To completely neutralize these misclassifications in the RAG pipeline, the system triggers Category-Filtered Semantic Search, ensuring retrieved reference examples are restricted specifically to the predicted class domain.
